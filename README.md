# مشروع كاشف الأخبار الكاذبة (Fake News Detector)

هذا المشروع عبارة عن تطبيق ويب بسيط يساعد في التحقق من مصداقية الأخبار العربية باستخدام تحليل الكلمات المفتاحية وفحص المصادر الموثوقة.

## متطلبات التشغيل
- خادم ويب يدعم PHP (مثل XAMPP أو WampServer).
- قاعدة بيانات MySQL.
- Node.js لتشغيل واجهة React.

## خطوات الإعداد

### 1. إعداد قاعدة البيانات
- قم بإنشاء قاعدة بيانات جديدة باسم `fake_news_db`.
- قم باستيراد الملف الموجود في `database/schema.sql`.

### 2. إعداد الخلفية (Backend)
- انسخ مجلد `backend` إلى مجلد `htdocs` في XAMPP (أو المجلد المقابل في خادمك).
- تأكد من إعدادات الاتصال في `backend/config/db.php`.

### 3. إعداد الواجهة الأمامية (Frontend)
- افتح المجلد `frontend` في اليونكس/تيرمينال.
- قم بتشغيل الأمر التالي لتثبيت المكتبات:
  ```bash
  npm install
  ```
- قم بتشغيل التطبيق باستخدام:
  ```bash
  npm run dev
  ```

## كيفية الاستخدام
1. افتح التطبيق في المتصفح.
2. الصق نص الخبر الذي تريد التحقق منه.
3. (اختياري) أضف رابط الخبر.
4. اضغط على زر "تحليل الخبر".
5. ستظهر لك النتيجة مع نسبة الثقة والأسباب.

## تقنية العمل
- **الواجهة الأمامية:** React 19, Vanilla CSS.
- **الخلفية:** Pure PHP.
- **قاعدة البيانات:** MySQL.
- **خوارزمية التقييم:** نظام يعتمد على القواعد (Rule-based) وليس الذكاء الاصطناعي المعقد، مما يجعله مثالياً لمشاريع التخرج التعليمية.
